# MedMentorIA

## Visão Geral
Plataforma de preparação para residência médica com IA preditiva. Oferece banco de questões comentadas, simulado diagnóstico (10q) e relatório preditivo de desempenho.

## Produto Principal
- **Simulado 10q**: diagnóstico inicial de 10 questões
- **Relatório Preditivo (RP)**: análise de lacunas por especialidade
- **Simulado 40q (RA)**: avaliação completa
- **Banco de questões comentadas**: diferencial principal de produto

## Público-alvo
Médicos se preparando para concursos de residência médica.

## Posicionamento
Diferencial frente a concorrentes (Aristo, Med Grupo, Estratégia Med): IA preditiva + simulado personalizado + relatório de lacunas.
